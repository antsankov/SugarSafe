What is SugarSafe?

The Series is a set of four workshops for students to learn skills like electronics, prototyping, graphic design, and internet of things.

Electronics

Learn concepts of electrical engineering along with technical skills like breadboarding, arduino microcontrollers, and soldering.

Prototyping

Learn concepts of prototyping along with technical skills like ideation, physical prototyping, solidworks, and 3D printing.

Design

Learn concepts of design along with technical skills like graphic design, web design, user interface design, and user experience design.

IOT

Learn the novel concept of Internet of Things along with a variety of technologies that make it possible like the Intel Edison.
